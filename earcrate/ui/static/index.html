    Setup

    Performance
    Lower is faster. 180s is plenty for BPM/key. Raise only if you have long DJ sets or mixes where the useful part starts late.
    Parallel decode + analysis across CPU cores. Auto uses all but two.
    

    

    Add music // multi-folder ingest
    

    Copy music from anywhere (external SSD, downloads, old drives) into the managed library. Sources are never modified; copies land under master/ingested/<batch>/, content-hash deduped, manifest-journaled, rollback-able. Then Organize + Retag builds the clean Artist/Album/NN Title tree with amended tags under working/organized/ — your originals stay verbatim.
