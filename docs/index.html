[ Install // three steps ]
  1. Click the button — it saves get-earcrate.cmd to your Downloads.
  2. Double-click the file. If Windows shows a blue "protected your PC" screen, click More info → Run anyway (it's your own file from your own repo).
  3. That's it. It installs Python + ffmpeg if you don't have them, downloads the latest EarCrate, builds it, puts an EarCrate icon on your Desktop, and opens the console. If it says it installed missing pieces, double-click the file one more time.
After it opens: in Setup pick your music folder + workspace. Then Library feeds and cleans it — Ingest (point it at a drive), Organize (builds your clean Artist/Album archive), Build ear crate (auditions your own music into mashup atoms). Then Play — pick a resident and Book a set. Dry-run is always the default; your source files are never touched.

To update later: run the same setup file again — it always installs whatever this button says it ships. Proof you're current: the console's command strip shows the same version + build stamp as the button above.
