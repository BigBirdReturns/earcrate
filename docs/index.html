earcrate

Local-first mashup engine. Your music never leaves your machine — this button just fetches a tiny setup file that does everything for you.

⬇ Download EarCrate Setup Windows · one double-click · currently ships __PAGE_BUILD__
  1. Click the button — it saves get-earcrate.cmd to your Downloads.
  2. Double-click the file. If Windows shows a blue "protected your PC" screen, click More info → Run anyway (it's your own file from your own repo).
  3. That's it. It installs Python + ffmpeg if you don't have them, downloads the latest EarCrate, builds it, puts an EarCrate icon on your Desktop, and opens the app. If it says it installed missing pieces, double-click the file one more time.
After it opens: pick a workspace folder, then use the Library tab — Ingest (point it at your external drive), Organize (builds your clean Artist/Album archive), Ear Crate (builds your mashup reference from your own music). Dry-run is always the default; your source files are never touched.

To update later: just run the same setup file again — it always installs whatever this button says it ships. Proof you're current: the app's header shows the same version + build stamp as the button above.
